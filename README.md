# hugolify-theme-2

## Install

Edit `config/_default/module.yaml` to install the `hugolify-theme-2` module:

```yml
imports:
  - path: github.com/hugolify/hugolify-theme-2
```

Add this import in `assets/sass/main.sass`

```sass
/*! purgecss start ignore */
// Theme
@import "hugolify-theme-2"
/*! purgecss end ignore */
```
